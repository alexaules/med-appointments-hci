# Escala SUS
## Instrucciones
Responder en escala Likert de 1 a 5.

1. Considero que me gustaría usar este sistema frecuentemente.
2. Encontré el sistema innecesariamente complejo.
3. Considero que el sistema es fácil de usar.
4. Creo que necesitaría apoyo técnico para usarlo.
5. Las funciones del sistema están bien integradas.
6. Encontré inconsistencias dentro del sistema.
7. Imagino que la mayoría de las personas aprenderían a usarlo rápido.
8. Encontré el sistema complicado de usar.
9. Me sentí confiado al usar el sistema.
10. Necesité aprender demasiadas cosas antes de usar el sistema.

## Resultado de ejemplo
Puntaje promedio estimado: 82/100
Interpretación: usabilidad alta.
